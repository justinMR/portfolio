# Fonts

Self-hosted Inter and Geist (variable first, static TTF fallback). Filenames must match `styles.scss`:

- `inter.ttf` — Inter variable (100–900)
- `inter-regular.ttf` — Inter 400 fallback
- `inter-medium.ttf` — Inter 500 fallback
- `geist.ttf` — Geist variable (100–900)
- `geist-semibold.ttf` — Geist 600 fallback

Sources: [Inter](https://rsms.me/inter/), [Geist](https://vercel.com/font)
